# Lab02: Unit Testing

## Overview

Simple ATM interactive screens with view balance, withdraw, deposit functionality. 

## Getting started

* Clone the repo;
* Build with `dotnet build --configuration Release`;
* Find the dll in `bin/Release` folder;
* Start with `dotnet ConsoleATM.dll`;
